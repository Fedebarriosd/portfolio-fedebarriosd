# portfolio-fede-tailwind

Starter Vite + React + Tailwind (glassmorphism) para Fede.

## Pasos
1. Instalar deps
```bash
npm install
```

2. Dev
```bash
npm run dev
```

3. Build
```bash
npm run build
```

Si tenés problemas con `npx`/`tailwindcss`, *no hace falta* correr `init`: ya están
`tailwind.config.js` y `postcss.config.js` listos.
